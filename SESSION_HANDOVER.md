### 第二十一阶段：新增"压力知觉测评"心理测验

**新增文件**：压力知觉测评.html - 压力知觉量表（PSS-10）自评工具

**量表来源**：
- 原始量表：Cohen, Kamarck, & Mermelstein (1983) 编制
- 中文版：赵鑫、杨小欢、丁文娟、陈睿、蒋鑫、张艳艳（2017）汉化
- 本研究：应用于癌症生存者群体验证信效度
- 发表期刊：《中国心理卫生杂志》2025年第39卷第5期
- 页码：369-374

**量表结构**：
- 10个条目，单维度结构
- 5点评分（0=从不，1=偶尔，2=有时，3=经常，4=总是）
- 总分范围：0-40分
- **有反向计分题**：第4、5、7、8题需要反向计分（score = 4 - score）

**信效度指标**：
- Cronbach's α = 0.86
- 重测信度 = 0.85
- CFI = 0.97, TLI = 0.96, RMSEA = 0.08

**评分标准（按分数分布划分）**：
- 低水平（0-13分）：压力知觉较低
- 中等水平（14-26分）：中等程度压力知觉
- 高水平（27-40分）：压力知觉较高
- 划分依据：按前27%、后27%及中间46%划分

**文献引用（APA 7格式）**：
- Cohen, S., Kamarck, T., & Mermelstein, R. (1983). A global measure of perceived stress. *Journal of Health and Social Behavior, 24*(4), 385-396.
- 赵鑫, 杨小欢, 丁文娟, 陈睿, 蒋鑫, 张艳艳. (2017). 压力知觉量表中文版在癌症患者群体中的信效度检验. *中国心理卫生杂志, 31*(10), 762-768.

## 技术实现细节

### 配色切换机制

```javascript
// localStorage存储用户偏好
localStorage.setItem('theme', 'light'); // 或 'dark'

// 页面加载时读取
const savedTheme = localStorage.getItem('theme');
if (savedTheme === 'light') {
    document.documentElement.setAttribute('data-theme', 'light');
}
```

```css
:root {
    --primary: #6B5CE7;
    --bg: #1A1B2E;

[data-theme="light"] {
    --primary: #5B4CD7;
    --bg: #F5F5FA;
    --card: #FFFFFF;
    --text: #2A2A3E;
    /* ... */
}
```
