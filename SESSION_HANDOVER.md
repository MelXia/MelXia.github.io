**游戏设计**：
- **第一轮（3试次）**：单人模式，熟悉规则。玩家vs虚拟对手，各有一张0-10随机数字卡片，数字大者获胜，赢+10代币，输-10代币
- **第二轮（5试次）**：合作者加入。玩家先看到自己的卡片，选择是否翻开合作者的卡片。翻开=绑定合作（双赢各+5，任一输各-5）；不翻开=单打独斗（赢+10，输-10）
- **第三轮（5试次）**：自由抉择模式。与第二轮规则相同，但翻开合作者底牌后可以自由选择合作或不合作（第二轮是翻开即绑定合作）

**核心心理学理论**：
- 社会困境（Social Dilemmas）：个人理性与集体理性的冲突
- 损失厌恶（Loss Aversion）：收益减半、风险减半触发本能反应
- 搭便车心理：少付出、多收益的短期策略

**技术特点**：
- 卡片式UI设计，遮挡卡片带问号，翻开后显示数字
- 三轮递进，体验信息不对称下的信任抉择
- 结果页面：柱状图显示各轮得分，洞察分析合作倾向
- 完整心理学科普页面（给家长）

## 技术实现细节

### 配色切换机制

```javascript
// localStorage存储用户偏好
localStorage.setItem('theme', 'light'); // 或 'dark'

// 页面加载时读取
const savedTheme = localStorage.getItem('theme');
if (savedTheme === 'light') {
    document.documentElement.setAttribute('data-theme', 'light');
}
```

### CSS变量定义

```css
:root {
    --primary: #6B5CE7;
    --bg: #1A1B2E;
    --card: #252742;
    --text: #E8E8F0;
    /* ... */
}

[data-theme="light"] {
    --primary: #5B4CD7;
    --bg: #F5F5FA;
    --card: #FFFFFF;
    --text: #2A2A3E;
    /* ... */
}
```

### 汉堡菜单实现

```javascript
function toggleMobileMenu() {
    const menu = document.getElementById('mobileMenu');
    const toggle = document.querySelector('.menu-toggle');
    menu.classList.toggle('show');
    toggle.classList.toggle('active');
}
```
